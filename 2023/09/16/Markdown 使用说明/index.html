这是一篇关于讲解如何正确使用Markdown的排版示例，希望通过此，大家都能轻松上手，都能通过Markdown 能够让自己的文章有更加出色、更清晰明了的排版。

什么是 Markdown

Markdown (MD) 是现在普遍使用的一种文档书写语言格式，只需用一些非常简单易记的符号，如（# * / > [] ()\ ），就可以轻松写出一篇具有良好的排版和可读性的文章。

6.链接

支持直接粘贴链接地址。链接语法和显示如下：

语法1：[链接名称](http://链接网址)

显示：链接名称

或语法2：<http://链接网址>

显示：http://链接网址

7. 图片

可通过上传按钮上传图片，也支持直接粘贴图片，也支持拖拽上传图片。图片语法和显示如下：

语法：![请添加链接描述](图片地址)

显示：![请添加链接描述](图片地址)

8. 有序列表&有序列表

无序列表的使用，在符号“-”后加空格使用。如下：

  • 列表项

  • 列表项

  • 列表项

如果要控制列表的层级，则需要在符号“-”前使用空格。如下：

  • 列表项

  • 列表项

    • 列表内容

    • 列表内容

有序列表的使用，在数字及符号“.”后加空格几个，如下：

  1. 列表项

  2. 列表项

    1. 列表内容

    2. 列表内容

      1. 子列表内容

      2. 子列表内容

      3. 子列表内容

9. 分割线

可以在一行中用三个以上的星号、减号、底线来建立一个分隔线，同时需要在分隔线的上面空一行。

例：

-------

*****

______

10. 符号转义

如果你的描述中需要用到 markdown 的符号，比如 # * 等，但又不想它被转义，这时候可以在这些符号前加反斜杠，如 \ # *进行避免。
